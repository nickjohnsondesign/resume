Tickets Case Study

Project Goal

To design an application that would help users find a movie at a local theater and reserve seats to make sure they make it to the big premiere and to make sure the design was user-centric and friendly, I've created personas and storyboards to help empathize with the user.

  • Challenges

  • Create an intuitive user journey
  • Design with accessibility in mind
  • Keep UI and visual design focused on user goals
  • Design a quick and painless checkout process
