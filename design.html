Web Design and Wireframes

Sketch.Prompt - React/JS based prompt generator

Sketch.Prompt is a React/JS-based prompt generator that provides artists and designers with daily creative inspiration. It generates a unique type/font/color with each refresh and a new prompt is generated every day.

Low Fidelity Wireframe - Movie Theater Seat Reservation App

This low fidelity wireframe showcases the user flow for a movie theater seat reservation app. It focuses on the initial layout and structure, providing a visual blueprint for the application's development and user journey.

High Fidelity Wireframe - Movie Theater Seat Reservation App

This is a high fidelity wireframe done in Figma to showcase the functionality for my mockup of a movie theater seating reservation app. The wireframe showcases unique visual design, careful attention to detail, interactive elements, and an understanding user-centric design philosophy.
